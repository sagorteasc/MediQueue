# 🎓 MediQueue

> A smart and modern tutor finding platform that helps students connect with the right tutors easily. MediQueue simplifies tutor discovery, session booking, and schedule management through a clean, responsive, and user-friendly experience.

---

## 🌐 Live Links

- 🔗 Live Site: https://medi-queue-wine.vercel.app
- 💻 GitHub: https://github.com/sagorteasc/MediQueue

---

# ✨ Features

## 👨‍🎓 Student Features

- 🔍 Search and discover tutors
- 📅 Book tutoring sessions
- 👤 View tutor profiles
- 📱 Responsive dashboard
- 🔐 Secure authentication

## 👨‍🏫 Tutor Features

- 📝 Create tutor profiles
- 📚 Add expertise & subjects
- ⏰ Manage schedules
- ✅ Accept or reject bookings

## 🛠️ Admin Features

- 👥 Manage users
- 📊 Monitor activities
- 🔒 Role-based access control
